# NodeJs Learnings

This repository collects multiple Node.js and Express learning projects in one GitHub repository.

## Repository structure

- `ExpressJs/`
- `expressjs-basics/`
- `Node-and-Express/`
- `node-server/`
- `Node-Vs-Express/`
- `nodejs/`
- `nodejs-expressjs-templating-engines/`
- `nodejs-mvc/`
- `nodejs-single-fileupload/`
- `nodejs_web_server/`
- `redis/`
- root-level practice files such as `server.js`, `fs.js`, and related notes

## Notes

- Each project keeps its own source code inside its folder.
- Dependency folders like `node_modules/` are ignored at the repository level.
- If a project needs packages after cloning, run `npm install` inside that specific project folder.

## Initial GitHub push

After creating an empty repository on GitHub, run:

```bash
git remote add origin <your-github-repo-url>
git branch -M main
git push -u origin main
```
